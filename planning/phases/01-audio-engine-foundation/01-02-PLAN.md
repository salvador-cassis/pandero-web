From @soundtouchjs/audio-worklet v1.0.8 (via importmap in poc.html):
```javascript
// Registration — once per AudioContext
await SoundTouchNode.register(audioCtx, processorUrl);  // processorUrl: string

// Construction
const stNode = new SoundTouchNode(audioCtx);

// AudioParams
stNode.pitch.value           // default 1.0 (multiplier, NOT used for key changes)
stNode.pitchSemitones.value  // default 0, range -24 to +24 (use this for key changes)
stNode.playbackRate.value    // USED in tandem pattern (not stNode.tempo)

// Connect like any AudioNode
stNode.connect(audioCtx.destination);
```

Tandem pattern for tempo (MANDATORY — prevents gap artifacts above 1.0x):
```javascript
source.playbackRate.value = ratio;  // feeds samples at the new rate
stNode.playbackRate.value = ratio;  // processor divides pitch correction by this
// stNode.tempo is NOT used in the tandem pattern
```

Pitch-only (independent of tempo):
```javascript
stNode.pitchSemitones.value = parseInt(semitones);  // source.playbackRate unchanged
```

AudioContext lifecycle:
```javascript
// Pause: suspend (preserves position)
await audioCtx.suspend();

// Resume: check both states (iOS uses 'interrupted')
if (audioCtx.state === 'suspended' || audioCtx.state === 'interrupted') {
  await audioCtx.resume();
}

// AudioBufferSourceNode is fire-and-forget — cannot be restarted after stop()
// On replay: create new source node, assign same audioBuffer, reconnect, start()
```

    - Wire graph:

      stNode = new SoundTouchNode(audioCtx);

      ```
    - Read current slider values and apply tandem pattern (ENG-02):

    - `await audioCtx.suspend()`   — preserves playback position, do NOT call source.stop()
    - `isPlaying = false`
    - Update `#status` to "Paused"
    - Disable `#pause-btn`, update `#play-btn` text to "Resume"

    **Tempo slider `input` handler:**
    - Guard: `if (!source || !stNode) return;`
    - `const ratio = parseFloat(e.target.value)`

    - Guard: `if (!stNode) return;`

    document.getElementById('play-btn').addEventListener('click', play);

    document.getElementById('tempo').addEventListener('input', handleTempoChange);
    document.getElementById('pitch').addEventListener('input', handlePitchChange);
    ```

    **Critical constraint — do NOT:**
    - Create AudioContext outside a user gesture handler

    - Set `source.playbackRate` without also setting `stNode.playbackRate` to the same value
