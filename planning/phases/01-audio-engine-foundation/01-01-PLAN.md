  </read_first>
  <action>
    Create `pandero/poc.html` with this exact content:

    ```html
    <!DOCTYPE html>
    <html lang="es">
    <head>
      <meta charset="UTF-8">
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
      <title>Pandero PoC — Phase 1</title>
      <script type="importmap">
      {
        "imports": {
          "@soundtouchjs/audio-worklet": "https://esm.sh/@soundtouchjs/audio-worklet@1.0.8",
          "unmute-ios-audio": "https://esm.sh/unmute-ios-audio@3.3.0"
        }
      }
      </script>
      <script>
        // Guard: AudioWorklet requires Secure Context (HTTPS or localhost).
        // Opening via file:// silently breaks addModule() with a cryptic DOMException.
        if (!window.isSecureContext) {
          document.write('<p style="color:red;font-weight:bold">ERROR: Must be served via HTTP (localhost) or HTTPS — not file://. Run: python3 -m http.server 8000</p>');
          throw new Error('Not a Secure Context. Halting.');
        }
      </script>
    </head>
    <body>
      <h2>Pandero PoC — Audio Engine Phase 1</h2>
      <p id="status">Not initialized</p>
      <button id="play-btn">Play</button>
      <button id="pause-btn" disabled>Pause</button>
      <br><br>
      <label>
        Tempo (ratio):
        <input id="tempo" type="range" min="0.5" max="1.5" step="0.01" value="1.0">
        <span id="tempo-val">1.00</span>
      </label>
      <br>
      <label>
        Pitch (semitones):
        <input id="pitch" type="range" min="-6" max="6" step="1" value="0">
        <span id="pitch-val">0</span>
      </label>
      <script type="module" src="./poc.js"></script>
    </body>
    </html>
    ```

    Key decisions:
    - `viewport` meta tag ensures readability on mobile during Phase 1 testing
    - `isSecureContext` guard uses `document.write` + `throw` so the error is unmissable (not buried in console)
    - Pause button starts `disabled`; poc.js enables it after Play is clicked
    - `<span>` elements for live value display (populated by poc.js in plan 01-02)
    - `type="module"` on the script tag activates importmap resolution
  </action>
