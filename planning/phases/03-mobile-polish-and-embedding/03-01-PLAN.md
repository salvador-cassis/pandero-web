**1. @font-face declaration:**
```css
@font-face {
  font-family: 'Caveat';
  src: url('./caveat-regular.woff2') format('woff2');
  font-weight: 400;
  font-style: normal;
  font-display: swap;
}
```

**2. CSS custom property token system scoped to `#pandero-player`** (per D-14, D-15):
```css
#pandero-player {
  --pandero-bg:         #f5eedc;   /* off-white / goatskin */
  --pandero-surface:    #3b1f0e;   /* dark brown / wood */
  --pandero-accent:     #c4702b;   /* warm ochre */
  --pandero-text:       #3b1f0e;   /* dark brown */
  --pandero-track:      #c4a882;   /* mid ochre for slider tracks */
  --pandero-font:       'Caveat', cursive;
  --pandero-size-label: 0.9rem;
  --pandero-size-value: 1.1rem;
  --pandero-hex-size:   120px;
  --pandero-gap:        1.25rem;
}
```

**3. Widget root layout** (`#pandero-player`) — mobile-first flexbox column (per D-09, UI-01):
- `display: flex; flex-direction: column; align-items: center;`
- `gap: var(--pandero-gap);`
- `padding: 1.5rem 1rem;`
- `background-color: var(--pandero-bg);`
- `max-width: 375px; width: 100%; box-sizing: border-box;`
- `font-family: var(--pandero-font); color: var(--pandero-text);`
- `margin: 0 auto;` (center in host page)

**4. Hexagon play button** (`.pandero-play-btn`) — per D-05, D-06, D-08:
- `width: var(--pandero-hex-size, 120px);`
- `aspect-ratio: 1 / 0.866;` (fallback) then `aspect-ratio: 1 / cos(30deg);` (modern override, both declarations)
- Handmade clip-path with vertex perturbation (per D-06):
  ```
  clip-path: polygon(51% 1%, 99% 24%, 100% 76%, 49% 99%, 1% 77%, 0% 25%);
  ```
- `background-color: var(--pandero-surface);`
- `color: var(--pandero-bg);`
- `border: none; cursor: pointer;`
- `font-size: 2rem;`
- `display: flex; align-items: center; justify-content: center;`
- `transition: opacity 0.15s ease;`
- `padding: 0;` (reset browser defaults)
- Active state: `.pandero-play-btn:active { opacity: 0.75; }`
- Touch target: the hexagon at 120px wide already exceeds 44px minimum

**5. Controls container** (`.pandero-controls`) — per D-09:
- `display: flex; flex-direction: column; gap: var(--pandero-gap); width: 100%;`

**6. Slider row** (`.pandero-slider-row`) — per D-11:
- `display: flex; flex-direction: column; gap: 0.25rem; width: 100%;`

**7. Slider header** (`.pandero-slider-header`) — label ABOVE slider per D-11:
- `display: flex; justify-content: space-between; align-items: baseline;`
- `font-family: var(--pandero-font); font-size: var(--pandero-size-label); color: var(--pandero-text);`

**8. Value display** (`.pandero-value`):
- `font-size: var(--pandero-size-value); font-weight: 400;`

**9. Range slider base** (`.pandero-slider`) — per UI-02:
- `-webkit-appearance: none; appearance: none;`
- `width: 100%; height: 44px;` (iOS touch row height for UI-02)
- `background: transparent; cursor: pointer; padding: 0; margin: 0;`

**10. WebKit slider thumb** (`.pandero-slider::-webkit-slider-thumb`) — circular per D-12, 44px touch per UI-02:
- `-webkit-appearance: none; appearance: none;`
- `width: 24px; height: 24px; border-radius: 50%;`
- `background-color: var(--pandero-accent);`
- `border: 10px solid transparent;` (expands hit area to 44px: 24 + 20 = 44)
- `box-sizing: content-box; background-clip: padding-box;`
- `margin-top: -20px;` (center on 4px track: -(44-4)/2 = -20)
- `cursor: pointer;`

**11. WebKit slider track** (`.pandero-slider::-webkit-slider-runnable-track`):
- `height: 4px; background: var(--pandero-track); border-radius: 2px;`

**12. Firefox slider thumb** (`.pandero-slider::-moz-range-thumb`) — separate rule, cannot combine with WebKit:
- `width: 24px; height: 24px; border-radius: 50%;`
- `background-color: var(--pandero-accent); border: none; cursor: pointer;`

**13. Firefox slider track** (`.pandero-slider::-moz-range-track`):
- `height: 4px; background: var(--pandero-track); border-radius: 2px;`

No reset button styles needed (per D-13 — reset removed from widget).
